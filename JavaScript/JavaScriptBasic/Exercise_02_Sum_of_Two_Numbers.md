# Exercise 02 :

### Sum of Two Numbers

- Create a function that takes two numbers as arguments and returns their sum.

Examples:

1. addition(3, 2) = 5
2. addition(-3, -6) = -9
3. addition(7, 3) = 10

Notes:
* Don't forgetto return the result.

-------------

### Solution:

```javascript
function addition( a, b ) {
	return a + b
}
```
